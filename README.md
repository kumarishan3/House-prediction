House Price Prediction (Machine Learning Project)

 Problem
 Predict house prices based on area and number of rooms.


 Dataset
The dataset contains:
- Area (square feet)
- Number of rooms
- Price

 Approach
- Loaded dataset from CSV file
- Performed exploratory data analysis (EDA)
- Checked correlation between features
- Applied feature scaling
- Trained Linear Regression model
- Evaluated model using MAE, RMSE, and R² score

 Results
- Strong linear relationship between area and price
- Achieved high R² score (~0.98)

 Key Learnings
- Regression modeling
- Feature importance
- Model evaluation techniques

  Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
